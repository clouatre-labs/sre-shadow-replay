Code snippet to demonstrate the bug
```
from sqlglot import parse_one

ddl = "CREATE TABLE tbl (id BIGINT NOT NULL GENERATED BY DEFAULT AS IDENTITY (START WITH 1 INCREMENT BY 1))"

ddl_parsed = parse_one(ddl, read="databricks")
sql = ddl_parsed.sql(dialect="databricks")
print(sql)
```

Before the fix:
`CREATE TABLE tbl (id BIGINT NOT NULL GENERATED ALWAYS AS IDENTITY (START WITH 1 INCREMENT BY 1))`

Expected:
`CREATE TABLE tbl (id BIGINT NOT NULL  BY DEFAULT AS IDENTITY AS IDENTITY (START WITH 1 INCREMENT BY 1))`
